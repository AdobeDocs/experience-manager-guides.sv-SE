# Anpassa redigeraren {#id181AI0003PN}

AEM Guides levereras med en kraftfull Web Editor med vilken dina författare kan skapa och redigera DITA-dokument. Du kan anpassa redigerarens verktygsfält för att lägga till eller ta bort funktioner som du kan komma åt från verktygsfältet. Du kan också konfigurera autosparfunktioner för filer, generera ID:n för element när de infogas i dina dokument och mycket annat.
