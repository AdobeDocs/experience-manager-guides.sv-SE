# Tekniska krav {#id213BCG003XA}

AEM Guides görs tillgängligt via Adobe Software Distribution Portal. Du kan hämta AEM Guides från ditt Adobe Software Distribution Portal-konto och installera det på alla Adobe Experience Manager-instanser (AEM) i din konfiguration. Vanligtvis lagras din redigeringsinstans och produktionsinstans av AEM på olika servrar. Du måste installera AEM Guides på alla instanser av AEM som du tänker använda.

Innan du installerar Adobe Experience Manager Guides för On-Premise måste du kontrollera att datorn uppfyller de tekniska krav som anges nedan för den aktuella versionen.

**Webbläsare som stöds**

- Google Chrome
