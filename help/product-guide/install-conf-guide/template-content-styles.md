# PDF Publishing

Med AEM Guides kan du generera PDF:er av enskilda ämnen eller en karta med den inbyggda funktionen för generering av utdata från PDF. Du kan anpassa och använda PDF-mallar när du genererar PDF-utdata. Du kan skapa och utforma sidlayouter för att presentera de olika avsnitten i PDF-utdata som du vill.

Dessutom har AEM Guides stöd för språkvariabler som hjälper dig att definiera lokaliserade strängar. Du kan också definiera format för olika element i PDF-utdata. Om du är en avancerad användare kan du även arbeta med CSS direkt för att skapa avancerade sidlayouter.
